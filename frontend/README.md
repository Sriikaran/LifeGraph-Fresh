# Kart.in - Outcome Commerce Platform

Kart.in is a next-generation ecommerce platform that shifts the paradigm from "transactional commerce" to "outcome commerce." Instead of just selling products, Kart.in uses intelligence to ensure users achieve the actual outcomes they desire (e.g., getting fit, eating healthier, sleeping better) by guiding their purchases, tracking their success, and offering verified mission pathways.

## Features

### 1. Outcome Commerce Overview
Kart.in goes beyond the traditional shopping cart. Every product can be tied to a specific life outcome. The platform intelligently bundles items that work together to guarantee results.

### 2. Demo Mode
A built-in interactive demonstration overlay perfect for hackathon judging and presentations. Demo Mode provides judges with real-time architectural insights, dynamic highlights, and interactive walkthroughs of the platform's core intelligence layer without breaking the marketplace immersion.

### 3. Mission Engine
Users can enroll in "Missions" (e.g., *The Deep Sleep Protocol*, *30-Day Fitness Challenge*). The Mission Engine tracks their purchases, validates prerequisites, and unlocks rewards as they achieve their goals.

### 4. Risk Engine
The platform dynamically assesses the "Risk" of a purchase failing to deliver an outcome. It analyzes cart synergy, user history, and product compatibility to provide real-time confidence scores before checkout.

## Future Vision

Our goal is to build an ecosystem where merchants are incentivized by user success rather than mere sales volume. Future iterations will include deeper AI personalization, IoT integration for real-world outcome verification (e.g., connecting to fitness trackers), and a decentralized reputation system for outcome-verified products.

## Setup Instructions

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```
